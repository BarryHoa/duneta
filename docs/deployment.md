# Deploy — Cloudflare Workers

```text
your-domain.com/*
  worker.ts
    /api/*  → Hono (@duneta/server)
    static  → env.ASSETS  (app/build/client)
    /*      → React Router SSR  (app/build/server)
```

## Commands

```bash
pnpm dev      # local :8787 (HMR)
pnpm build    # sync + build app/
pnpm deploy   # build + wrangler deploy
```

## Production checklist

- [ ] **`duneta.server.config.ts`** — `process.env.*` refs (không literal secrets)
- [ ] **`wrangler secret put`** — DATABASE_URL, AUTH_SECRET, CSRF_SECRET
- [ ] **`secrets.required`** trong `wrangler.jsonc`
- [ ] **CI build sạch** — không `.env` / secret env vars lúc `pnpm build`
- [ ] **Hyperdrive** (optional) — connection string từ Hyperdrive → paste vào `database.connections.*.url`
- [ ] **Optional** — Redis HTTP URL trong `cache` config (distributed cache + rate limit)
- [ ] **Assets** — `ASSETS` binding → `app/build/client` (included in generated deploy config)
- [ ] **Logging** — JSON stdout (`logging.enabled` in `duneta.server.config.ts`)
- [ ] **Auth cookies** — `secure: true` applied automatically when `NODE_ENV=production`

## wrangler configs

| File | Purpose |
|------|---------|
| `wrangler.jsonc` | Dev + Vite build (`main: worker.ts`) |
| `wrangler.production.jsonc.example` | Reference template (Hyperdrive + ASSETS) |
| `app/build/server/wrangler.json` | Generated by `pnpm build` — used by `pnpm deploy` |

Example production bindings (`wrangler.production.jsonc.example`):

```jsonc
{
  "name": "duneta",
  "main": "worker.ts",
  "compatibility_flags": ["nodejs_compat"],
  "assets": {
    "directory": "app/build/client",
    "binding": "ASSETS",
    "run_worker_first": true
  },
  "hyperdrive": [
    { "binding": "HYPERDRIVE", "id": "<your-hyperdrive-config-id>" }
  ]
}
```

## Config

Secrets: `wrangler secret put` → runtime `process.env` → `duneta.server.config.ts`. Web: `duneta.client.config.ts` only.

## Observability

Workers have **no persistent filesystem**. Logs go to stdout → Wrangler tail / Dashboard / Logpush.

Request logs (when `logging.enabled`):

```json
{"level":"info","msg":"request","ts":"...","requestId":"...","method":"GET","path":"/api/health","status":200,"durationMs":12}
```
